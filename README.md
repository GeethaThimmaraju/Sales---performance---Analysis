# Sales Performance Analysis

## Project Overview
This project analyzes sales data to understand overall business performance across regions, product categories, customer segments, and time. The goal is to generate meaningful insights that support data-driven business decisions.

## Business Objectives
- Identify top-performing regions
- Analyze sales contribution by product category
- Understand customer segment performance
- Observe monthly sales trends

## Dataset
- Sales dataset containing order, customer, product, and sales information

## Tools & Technologies
- Excel (Pivot Tables)
- SQL (MySQL)
- Python (Pandas)

## Key Analysis Performed
- Cleaned raw data by handling missing values and converting date formats
- Performed region-wise, category-wise, and segment-wise sales analysis
- Conducted monthly sales trend analysis
- Validated insights across Excel, SQL, and Python

## Key Insights
- West region contributes the highest sales
- Technology category generates maximum revenue
- Consumer segment accounts for the largest share of sales
- Monthly sales show seasonal variations

## Files in This Repository
- sales_analysis.ipynb — Python analysis
- sales_analysis.sql — SQL queries
- Dataset file (if applicable)

## Conclusion
This project demonstrates an end-to-end data analysis workflow, from data cleaning to insight generation, using multiple analytical tools.
